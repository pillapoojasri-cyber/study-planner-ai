# Study Planner AI

A Flask-based Study Planner with AI-powered study guidance.

## Features

* User Login & Signup
* Forgot Password
* Add, Edit, Delete Tasks
* Complete Tasks
* Study Streak Tracking
* Progress Dashboard
* Pie Chart Analytics
* AI Study Coach using Groq API

## Technologies Used

* Python
* Flask
* SQLite
* HTML
* CSS
* JavaScript
* Chart.js
* Groq API

## Installation

```bash
pip install -r requirements.txt
python app.py
```
